# AI Mission Control

Standalone read-only operations dashboard for visualizing project activity.

It uses mock data only. It does not connect to any production application,
database, authentication provider, production API, or deployment system.

## Visualized Signals

- Git commits
- Modified files
- Passing tests
- Preview and production deployment status
- AI task activity
- Build pipeline stages
- Project structure

## Run Locally

```bash
cd ai-mission-control
npm start
```

Then open:

```text
http://localhost:4173
```
